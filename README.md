# 德语语音教程（仅供个人学习使用） 
**German Pronunciation Tutorial**

一个交互式德语语音学习网站，帮助你掌握德语的发音规则、长短元音、辅音组合、元音组合以及特殊发音。

👉 **在线体验**：https://ovo0902.github.io/German-Pronunciation/

---

## ✨ 主要功能

- **点击即听**：点击任意德语单词或例句，即可播放标准发音。
- **录音对比**：点击 🎤 图标录制自己的发音，与原音进行对比练习。
- **键盘快捷键**（超级方便）：
  - **WASD** 或 **方向键**：切换单词
  - **空格键**：播放当前发音
  - **Enter**：开始/停止录音
- **播放速度调节**：支持 0.8× 等速度，方便慢速跟读。
- **语音测试**：专门的测试表格，一键朗读整行单词。

---

## 📖 内容概览

网站按发音类别组织了丰富的表格和例句，包括：

1. **主要长短音**（元音 a, e, i, o, u 等）
2. **字母表**（Alphabet）
3. **缩略词**（常见缩写发音）
4. **元音组合**（ei, au, eu, ie, äu 等）
5. **辅音组合**（ch, sch, pf, ng, r 的不同发音等）
6. **语音测试**（综合练习）

## 🎯 适合人群

- 德语零基础或初学者（重点打好发音基础）
- 德语教师（可作为课堂演示工具）

---

## 📱 使用方法

1. 打开网站后，直接用鼠标点击文字听音。
2. 点击 🎤 开始录音练习（浏览器会请求麦克风权限）。
3. 使用键盘快捷键快速浏览和练习。
4. 建议先慢速（0.8×）跟读，熟悉后再恢复正常语速。
